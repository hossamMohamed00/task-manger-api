# Task-Manger-Api
This is a simple Task Manger Api used to track users tasks

## Live Demo
Test the apis from postman 
https://hossam-task-manger.herokuapp.com/users/me

## Technologies
- Nodejs
- Express
- Mongoose
- Rest apis

## Installation

Use the package manager [npm](npmjs.com) to install all dependencies.

```bash
npm install 
```
## Usage
To run this project locally on your machine
```bash
npm run dev
```
